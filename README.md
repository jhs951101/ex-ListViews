(1) react native 프로젝트 생성
  1) expo init HelloWorld <br/>
  2) cd HelloWorld <br/>
  3) yarn start <br/>


(2) 프로젝트 다운로드 후 실행
  1) yarn add expo <br/>
  2) (필요한 라이브러리 모두 다운로드) <br/>
  3) yarn start <br/>


(3) React Native 프로젝트를 Android & iOS 프로젝트로 변환
  1) (node_modules 폴더가 없을 경우) yarn add expo <br/>
  2) (필요한 라이브러리가 없을 경우 모두 다운로드) <br/>
  3) npm run eject <br/>
  4) yarn android <br/>
